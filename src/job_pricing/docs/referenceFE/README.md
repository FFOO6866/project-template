# Talent Verse Dashboard

Talent and Organisation Performance Dashboard for Tsao Pao Chee (TPC).

## Project Overview

Talent Verse is an innovative, visually engaging, and strategically insightful Talent and Organisation Performance Dashboard for TPC. This dashboard empowers stakeholders (from Chairman to Head of People & Organisation) with actionable insights derived from rich individual, team, and organisational data, psychometric assessments, and AI-driven analytics.

## Strategic Outcomes

- Provide clear visibility of organisational performance across dimensions: People, Process, and Performance.
- Align HR and talent management initiatives with business outcomes and strategic pillars (Business, Investment, Advocacy, Philanthropy).
- Enable evidence-based, personalised interventions to drive performance, engagement, and growth.
- Enhance decision-making through real-time, AI-generated insights and recommendations.

## Technology Stack

- **Frontend**: Next.js, React, Tailwind CSS, shadcn/ui
- **Visualizations**: Recharts, D3.js
- **Animations**: Framer Motion
- **Architecture**: Feature-based organization with clean architecture principles

## Development Guidelines

- Organize code by feature, not by technical layer
- Each feature module should be self-contained
- Maintain low coupling and high cohesion across features
- Implement progressive delivery: only one feature/module to be built at a time
